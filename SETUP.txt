========================================
  EventBot Pro — Setup & Usage Guide
========================================

FIRST TIME SETUP
----------------
1. Install dependencies:
   pip install -r requirements.txt

2. Seed existing data (already done):
   python seed_from_excel.py

3. Clean up database:
   python fix_db.py


HOW TO USE
----------

A) START THE DASHBOARD (recommended)
   Double-click: start_dashboard.bat
   Opens in browser at: http://localhost:8501
   Password: workrbee2026

B) RUN THE BOT ONCE (manual)
   Double-click: run_bot_now.bat
   Scrapes all venues, saves Excel to output/ folder

C) AUTO-SCHEDULE (daily at 6 AM)
   Double-click: start_scheduler.bat
   Keep window open — bot runs automatically every day


DASHBOARD PAGES
---------------
  Dashboard     — Summary stats and recent events
  Run Bot       — Launch a scrape (select venues + year range)
  Live Tracker  — View/edit all records, log call notes, update status
  Export Excel  — Download full workbook (one tab per venue)
  Settings      — Venue list, add new venue, system info


OUTPUT FILES
------------
  output/EventBot_Tracker_YYYYMMDD_HHMM.xlsx
    - SUMMARY tab
    - One tab per venue
    - Columns: Event Name, Contact Person, Email, Telephone,
               Date of Event, e-mail sent, Call Notes 1-11
    - Sorted chronologically
    - Color-coded by status


ADDING NEW VENUES
-----------------
  Go to Settings > Add a New Venue in the dashboard.
  Copy the generated code block into venues.py and restart.


CHANGING PASSWORD
-----------------
  Edit start_dashboard.bat and change: workrbee2026
  Or set environment variable: EVENTBOT_PASSWORD=yourpassword


DATABASE LOCATION
-----------------
  data/events_db.json — all scraped data
  data/scheduler_config.json — scheduler settings
  logs/scheduler.log — run history
